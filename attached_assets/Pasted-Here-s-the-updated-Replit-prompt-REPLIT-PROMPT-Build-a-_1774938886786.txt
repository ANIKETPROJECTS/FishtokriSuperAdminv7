## Hub Management Page (Full CRUD)

Each Hub has:
- `name` (string, required) — e.g. "Thane Hub"
- `location` / address (string)
- `serviceAreas` (array of strings) — e.g. ["Mulund", "Bhandup", "Vikhroli"]
- `status` — Active / Inactive
- `createdAt` (auto)

### Hub Cards Layout
- Display hubs as **large, clean cards** in a responsive grid (3 columns desktop, 2 tablet, 1 mobile)
- Each card:
  - **AI-generated 3D warehouse/hub building illustration** at the top of the card as a card header image — vivid, glossy 3D render
  - Hub name in bold dark navy
  - Location in grey below
  - Service area tags as small blue pill badges (`#EFF6FF` background, `#1A56DB` text)
  - Status badge: green pill for Active, red pill for Inactive
  - Bottom row: **Edit** button (blue outline), **Delete** button (red outline), **Toggle Status** switch
  - Soft box shadow, rounded corners (`rounded-2xl`), white background
  - Hover: card lifts slightly with enhanced shadow

### Add Hub Button
- Top right of the page: solid blue button ("+ Add Hub") with a **3D AI-generated plus/add icon**
- Clicking opens a modal

### Add / Edit Modal
- Centered modal, white background, rounded corners, subtle shadow overlay
- Modal header: "Add New Hub" or "Edit Hub" in dark navy bold, with a small **3D AI-generated hub icon**
- Input fields: rounded, blue focus border, labels in dark navy
- Service Areas: tag-input style — type and press Enter to add tags, each tag shown as a blue pill with an × to remove
- Status toggle switch (blue when active)
- Footer buttons: "Cancel" (grey outline) and "Save Hub" (solid blue)

### Delete Confirmation Dialog
- Small centered dialog
- **AI-generated 3D warning/alert icon** (red exclamation in 3D style) at the top
- "Are you sure you want to delete this hub?" text
- "Cancel" (grey) and "Delete" (solid red) buttons
